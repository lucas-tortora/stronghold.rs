Expand description

Interface for the stronghold-p2p library to create a swarm, handle events and perform operations.

All Swarm interaction takes place in a separate task. StrongholdP2p is essentially a wrapper for the Sender side of a mpsc channel, which is used to initiate operations on the swarm.

Refer to StrongholdP2pBuilder for more information on the default configuration.
